LESSON 5.1: CSS List Style Types

Lists in HTML can have different bullet types or numbering styles. These can be customized with the `list-style-type` property.

Common List-Style Types:
  • list-style-type: disc; - Default bullet style (solid circle)
  • list-style-type: circle; - Hollow circle bullet
  • list-style-type: square; - Square bullet
  • list-style-type: decimal; - Numbered list (1, 2, 3...)
  • list-style-type: decimal-leading-zero; - Numbered list with leading zeros (01, 02, 03...)
        <ul>
          <li style="list-style-type:disc">Disc Bullet</li>
          <li style="list-style-type:circle">Circle Bullet</li>
          <li style="list-style-type:square">Square Bullet</li>
          <li style="list-style-type:decimal">Numbered List</li>
          <li style="list-style-type:decimal-leading-zero">Numbered List with Leading Zeros</li>
        </ul>

Custom Bullet Colors

To customize the color of list bullets, you can remove the default bullets and add a HTML entity that resembles a bullet using the • character.

        ul {
          list-style: none;
          padding: 0;
        }

        li {
          padding-left: 16px;
        }

CSS List-Style-Image Property

You can also replace bullets with images using the list-style-image property.

        ul {
          list-style-image: url('sqpurple.gif');
        }

Output:

CSS List-Style-Position Property

The list-style-position property specifies whether the bullet is inside or outside the list item content.

        ul.a {
          list-style-position: outside;
        }

        ul.b {
          list-style-position: inside;
        }

LESSON 4.3: CSS Table Styling

Table Borders

To specify table borders, use the border property.

        table, th, td {
          border: 1px solid;
        }

Output:

Full-Width Table

To create a full-width table, use the width: 100%; property on the <table> element.

        table {
          width: 100%;
        }

Output:

Collapse Table Borders

To collapse table borders into a single border, use the border-collapse property.

        table {
          border-collapse: collapse;
        }

LESSON 4.4: CSS Link Styling

Styling Links

Links can be styled using CSS properties such as color, background-color, and more.

        a {
          color: hotpink;
        }

Output:

Link Buttons

To style links as buttons, combine several CSS properties.

        a:link, a:visited {
          background-color: #f44336;
          color: white;
          padding: 14px 25px;
          text-align: center;
          text-decoration: none;
          display: inline-block;
        }

        a:hover, a:active {
          background-color: red;
        }
