Lesson 2.2: Color Properties

Color Property

The color property specifies the color of text.

Example of Color Palette with Coding:

<style>
 p.red { color: red; }
 p.blue { color: blue; }
 p.green { color: green; }
</style>
<body>
 <p class="red">This is red text.</p>
 <p class="blue">This is blue text.</p>
 <p class="green">This is green text.</p>

Lesson 2.3: Text Properties

Text Align Property

The text-align property specifies the horizontal alignment of text in an element.

<style>
 div.a { text-align: center; }
 div.b { text-align: left; }
 div.c { text-align: right; }
 div.d { text-align: justify; }
</style>
<body>
 <h1>The text-align Property</h1>
 <div class="a">
   <h2>text-align: center:</h2>
   <p>PUT YOUR NAME</p>
 </div>
 <div class="b">
   <h2>text-align: left:</h2>
   <p>PUT YOUR NAME</p>
 </div>
 <div class="c">
   <h2>text-align: right:</h2>
   <p>PUT YOUR NAME</p>
 </div>
 <div class="d">
   <h2>text-align: justify:</h2>
   <p>PUT YOUR NAME</p>
 </div>

Text Decoration Property

The text-decoration property specifies the decoration added to text.

<style>
 h1 { text-decoration: overline; }
 h2 { text-decoration: line-through; }
 h3 { text-decoration: underline; }
 h4 { text-decoration: underline overline; }
</style>
</body>
 <h1>PUT YOUR NAME 1</h1>
 <h2>PUT YOUR NAME 2</h2>
 <h3>PUT YOUR NAME 3</h3>
 <h4>PUT YOUR NAME 4</h4>

Output:

Text Indent Property

The text-indent property specifies the indentation of the first line in a text-block.

<style>
 div.a { text-indent: 150px; }
 div.b { text-indent: 5em; }
 div.c { text-indent: 10%; }
</style>
<body>
 <h1>The text-indent Property</h1>
 <div class="a">PUT YOUR NAME</div>
 <div class="b">PUT YOUR NAME</div>
 <div class="c">PUT YOUR NAME</div>
