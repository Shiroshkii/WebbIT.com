Lesson 1.1: WHAT IS HTML?

The Great Carrot Quest: An HTML Adventure In a lush green meadow, there lived a clever rabbit named Webbit. One sunny morning, Webbit decided he needed to find his favorite food: carrots. Just like the structure of a webpage, Webbit knew that his journey would have HTML elements guiding his along the way.

Webbit thought to himself, "Just like HTML provides structure to a website, I need a plan to find my carrots." He envisioned a map with specific paths leading to each delightful carrot patch, much like how HTML organizes content on a page.

HTML stands for Hypertext Markup language and is used in websites to structure their content. Think of it as a document that your browser reads to display a webpage. Every website relies on HTML, but each uses it differently. For example, Google Docs uses HTML alongside other coding languages like JavaScript in order for it to function. HTML is always working behind the scenes, playing a crucial role in making everything look and work the way it does.

Tim Berners-Lee wrote the initial HTML version in 1993. There have been numerous variations of HTML since that time. HTML 4.01 was the most commonly utilized version in the 2000's and was officially declared a standard in December 1999.

Example of Basic HTML Document

        <!DOCTYPE html>
        <html>
        <head>
        <title>My First Webpage</title>
        </head>
        <body>
        <h1>Welcome to My Webpage</h1>
        <p>This is a paragraph.</p>
        </body>
        </html>
