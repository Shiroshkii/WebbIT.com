Lesson 3.2: HTML STYLES

HTML STYLES

The HTML style is the attribute to add some style to your website, such as color, font, size, and more.

<html>
  <body>
    <h1>This is a paragraph</h1>
    <h2 style="color:blue;">This is a paragraph</h2>
    <h3 style="color:red;">This is a paragraph</h3>
  </body>
</html>

Output:

Background Color

The CSS background color property defines the background color for an HTML element.

<html>
 <body style="background-color:#D3D3D3;">
   <h1>This is a paragraph</h1>
   <h2 style="color:blue;">This is a paragraph</h2>
   <h3 style="color:red;">This is a paragraph</h3>
 </body>
</html>

Output:

Background font color

The CSS BackGround Color is like the highlighter at the back of the font.

<html>
 <body>
   <h1 style="background-color:#D3D3D3;">This is a paragraph</h1>
 </body>
</html>

Text Color

The CSS text color property defines the text color for an HTML element.

 <html>
   <body>
     <h1 style="color:gray">This is a paragraph</h1>
   </body>

Fonts

The CSS font-family property defines the font to be used for an HTML element

<html>
 <body>
   <h1 style="font-family:courier;">This is a paragraph</h1>
 </body>
</html>

Output:

Font Size

The CSS font-size property defines the text size for an HTML element

<html>
  <body>
    <h1 style="font-size:100;">This is a paragraph</h1>
  </body>
</html>

Text Alignment

The CSS text-align property defines the horizontal text alignment for an HTML element

<html>
  <body>
    <h1 style="text-align:center;">This is a paragraph</h1>
  </body>
