Lesson 5.2: Customizing the Mouse Cursor

Customizing the mouse cursor in CSS can enhance user experience by providing visual feedback or aligning with the design of your website. You can use the cursor property to specify different cursor styles.

Sample HTML Code for Different Cursors:

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
body {
  font-family: Arial, sans-serif;
  padding: 20px;
}
.default-cursor { cursor: default; }
.pointer-cursor { cursor: pointer; }
.crosshair-cursor { cursor: crosshair; }
.text-cursor { cursor: text; }
.not-allowed-cursor { cursor: not-allowed; }
.wait-cursor { cursor: wait; }
.custom-cursor { cursor: url('https://www.example.com/custom-cursor.png') 16 16, auto; }
.zoom-in-cursor { cursor: zoom-in; }
.zoom-out-cursor { cursor: zoom-out; }
.help-cursor { cursor: help; }
.move-cursor { cursor: move; }
.col-resize-cursor { cursor: col-resize; }
.row-resize-cursor { cursor: row-resize; }
.n-resize-cursor { cursor: n-resize; }
.e-resize-cursor { cursor: e-resize; }
</style>
</head>
<body>
  <div class="default-cursor">Default Cursor (default)</div>
  <div class="pointer-cursor">Pointer Cursor (pointer)</div>
  <div class="crosshair-cursor">Crosshair Cursor (crosshair)</div>
  <div class="text-cursor">Text Cursor (text)</div>
  <div class="not-allowed-cursor">Not Allowed Cursor (not-allowed)</div>
  <div class="wait-cursor">Wait Cursor (wait)</div>
  <div class="custom-cursor">Custom Cursor (custom)</div>
  <div class="zoom-in-cursor">Zoom In Cursor (zoom-in)</div>
  <div class="zoom-out-cursor">Zoom Out Cursor (zoom-out)</div>
  <div class="help-cursor">Help Cursor (help)</div>
  <div class="move-cursor">Move Cursor (move)</div>
  <div class="col-resize-cursor">Column Resize Cursor (col-resize)</div>
  <div class="row-resize-cursor">Row Resize Cursor (row-resize)</div>
  <div class="n-resize-cursor">North Resize Cursor (n-resize)</div>
  <div class="e-resize-cursor">East Resize Cursor (e-resize)</div>
</body>
</html>

Description of Each Style:

  1. Default Cursor (default): The standard arrow pointer, used when no specific style is applied.
  2. Pointer Cursor (pointer): A hand icon, indicating clickable elements, often used for links.
  3. Crosshair Cursor (crosshair): A crosshair, used in applications involving graphics or precise selections.
  4. Text Cursor (text): The "I-beam" cursor, indicating where text can be entered or selected.
  5. Not-Allowed Cursor (not-allowed): A circle with a line through it, showing that an action is not permitted.
  6. Wait Cursor (wait): An hourglass or spinning wheel, indicating that the user should wait.
  7. Custom Cursor (url): A custom image as the cursor. Replace the URL with your image link. The numbers indicate the cursor's hotspot.
  8. Zoom In Cursor (zoom-in): Suggests the ability to zoom in on an element.
  9. Zoom Out Cursor (zoom-out): Suggests the ability to zoom out on an element.
  10. Help Cursor (help): A question mark or icon suggesting help or additional information is available.
  11. Move Cursor (move): Indicates an element can be moved.
  12. Column Resize Cursor (col-resize): Shows that the user can resize a column, often seen in table headers.
  13. Row Resize Cursor (row-resize): Shows that the user can resize a row.
  14. North Resize Cursor (n-resize): Indicates resizing from the north (upward).
  15. East Resize Cursor (e-resize): Indicates resizing from the east (rightward).

Note:

To see the output, copy and paste the code into an HTML file and open it in your browser. Hover over each section to see the different cursor styles in action.
