LESSON 4.1: HTML COLORS

HTML colors can be provided using color names, Hex, Values, RGB, or HSL.

Color Names

When choosing the background or text color, you can directly give a color name. HTML provides a wide range of standard color names recognized by the majority of browsers, allowing you to choose freely

<html>
  <body>
    <h1 style="background-color:Crimson">Crimson</h1>
  </body>
</html>

Output:

Hex Values

If you desire a wider range of color options, Hex color codes are the solution. The Hex Codes begin with a hashtag (#), followed by a six-digit code that represents a color. The characters are represented by hexadecimal values that range from 00 to FF.

For example, set the red parameter to the highest value, FF to make them red, while setting the remaining two to the lowest value, 00. Like this: #FF000

If you want white, change all color parameters to FF, like this: #FFFFFF, to display white

If you want black, change all color parameters to 00, like this: #000000, to display black

<html>
 <body>
   <h1 style="background-color:#FF0000">#FF0000</h1>
   <h1 style="background-color:#FFFFFF">#FFFFFF</h1>
   <h1 style="background-color:#000000">#000000</h1>
 </body>
</html>

Output:

RGB Values

RRGB represents Red, Green, and Blue. The color intensity is defined by the parameters red, green, and blue, which range from zero to 255.

For example, set the green parameter to 255 and the other two colors to zero to get a particular green. Like this: RGB(0, 255, 0)

If you want white, all color parameters must be set to 255 to get white, like this: RGB(255, 255, 255), to display white

If you want black, all color parameters must be set to 0 to get black, like this: RGB(0, 0, 0), to display black

<html>
<body>
  <h1 style="background-color:rgb(0, 255, 0)">Green</h1>
  <h1 style="background-color:rgb(255, 255, 255)">White</h1>
  <h1 style="background-color:rgb(0, 0, 0)">Black</h1>
</body>
</html>

Output:

HSL Values

HSL represents Hue, Saturation, and Lightness Hue describes a color’s angle on the RGB color wheel, Red is produced by a color of 0°, green by 120°, and blue by 240° Saturation is the intensity of the hue. 100% represents the entire color, whereas 0% is a gray shade Lightness refers to the brightness of a color space. 0% for black and 100% for white. 

<html>
  <body>
    <h1 style="background-color:hsl(0, 0%, 65%)">hsl(0, 0%, 65%)</h1>
  </body>
</html>
