Lesson 2.1 Paragraphs, Headlines, and Text

The <P> Element
The <P> tag is used to create paragraphs in an HTML document. Each section starts and ends with the <p> tags. 

<html>
 <body>
   <p>This is a paragraph.</p>
   <p>This is another paragraph.</p>
 </body>
</html>

output:

Single Line Break

To add a line break without starting a new paragraph, use the <br> tag.

 <html>
  <body>
    <p>This is a paragraph. <br /> This is another paragraph</p>
  </body>
</html>
