Lesson 5.1: CSS Links Text Decoration

Text Decoration Property

In CSS, the text-decoration property is used to set the decoration of text in an element. When dealing with links, this property is commonly used to control how the text appears in terms of underlines, overlines, and strikethroughs.

Adding Underline to a Text:

<style>
a {
  text-decoration: underline;
}
</style>
<body>
  <a href="#">This link is underlined.</a>
</body>

Output:

This link is underlined.

Applying Overline, Line-Through, and Underline:

<style>
a {
  text-decoration: overline line-through underline;
}
</style>
<body>
  <a href="#">This link has multiple decorations.</a>
</body>

Output:

This link has multiple decorations.

Customizing Text Color:

<style>
a {
  text-decoration: underline;
  color: blue;
}
</style>
<body>
  <a href="#">This is a colored and underlined link.</a>
</body>

Output:

This is a colored and underlined link.

Customizing Text Decoration Style:

<style>
a {
  text-decoration: underline;
  text-decoration-style: dashed;
}
</style>
<body>
  <a href="#">This link has a dashed underline.</a>
</body>

Output:

This link has a dashed underline.

Customizing Text Decoration Thickness:

<style>
a {
  text-decoration: underline;
  text-decoration-thickness: 3px;
}
</style>
<body>
  <a href="#">This link has a thick underline.</a>
</body>

Output:

This link has a thick underline.

Combining Multiple Properties:

You can combine text-decoration with other related properties for even more control:

<style>
a {
  text-decoration: underline dotted;
  text-decoration-color: red;
  text-decoration-thickness: 2px;
}
</style>
<body>
  <a href="#">This link combines multiple properties.</a>
</body>

Output:

This link combines multiple properties.
